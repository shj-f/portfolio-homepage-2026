# Portfolio Website

웹 개발 및 퍼블리싱 작업 경험을 정리한 개인 포트폴리오 웹사이트입니다.

실제 프로젝트 결과물, CMS 연동 경험, 반응형 UI 구현 및 문제 해결 기록을 원페이지 형식으로 구성했습니다.

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Process

1. 이력서용 포트폴리오 목적에 맞춰 Hero, About, Skills, Projects, Work Log, Contact 구조를 설계했습니다.
2. Figma Make를 활용해 초기 UI 레이아웃과 디자인 시안을 생성했습니다.
3. 생성된 시안을 검토한 뒤, 원하는 방향과 실제 경력 범위에 맞도록 프롬프트를 반복 작성하며 레이아웃, 콘텐츠, 기술 표현, UI 구조를 조정했습니다.
4. Projects에서는 결과물과 담당 역할을, Work Log에서는 작업 중 발생한 문제 해결 경험을 분리해 구성했습니다.
5. React 기반으로 구현 후 빌드하여 GitHub Pages에 배포했습니다.

## AI Tools Used

- **ChatGPT**  
  포트폴리오 구조 설계, 섹션별 콘텐츠 정리, 기술 표현 검토, README 작성 보조

- **Figma Make**  
  초기 UI 레이아웃 및 디자인 시안 생성

- **Cursor / Claude Code**  
  React 코드 구조 확인 및 구현 보조

## AI Usage

AI는 초기 디자인 탐색, 정보 구조 정리, 문구 초안 및 코드 구현 보조 도구로 활용했습니다.

최종 포트폴리오 방향, 실제 프로젝트 내용 검토, 기술 범위 표현, 디자인 수정, 빌드 및 배포는 직접 진행했습니다.
